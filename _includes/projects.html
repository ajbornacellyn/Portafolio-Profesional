CryptoUnal

  • Crypto-wallet

CryptoUnal is a cryptocurrency wallet application that facilitates transactions of a particular cryptocurrency. The application uses Web3.js for interacting with the Ethereum blockchain. The frontend is developed using EJS, a templating engine for generating HTML with JavaScript. The backend is built using Express.js, a popular Node.js framework. For testing, Ganache-cli is used, which allows for a local blockchain to be set up for development purposes. The application follows a two-layer architecture, separating the frontend and backend logic.

PawsomeNetwork

  • microservices architecture

Pawsome Network is a social network dedicated to animal rescue, utilizing a microservices architecture with various programming languages such as Python, Go, Java, and Nest. It employs an API Gateway to facilitate communication between microservices and client requests, providing a comprehensive resource for pet owners and animal enthusiasts to share information on lost and found pets and animal welfare campaigns, encouraging responsible pet ownership.
